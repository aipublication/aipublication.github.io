When $a$ and $b$ are real numbers with $a<b$.

$$[a, b] = \{ x\ |\ a \le x \le b \}$$
$$[a, b) = \{ x\ |\ a \le x < b \}$$
$$(a, b] = \{ x\ |\ a < x \le b \}$$
$$(a, b) = \{ x\ |\ a < x < b \}$$
