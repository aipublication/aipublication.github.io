Power set of set $S$ is the set of all subsets of $S$.

































































































































































































































































































































































It is denoted as $P(S)$.


































































































































































































































































































































































If the [Cardinality]({% post_url 2020-07-09-cardinality %}) of set $S$ is n. Then size of power set will be $2^n$.
