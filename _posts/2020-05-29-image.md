Let $f: A \rightarrow B$ and $S \subset A$.
*Image* of set $S$ is the set of images (elements in [[Codomain]] mapped to elements in set $S$) in set $B$.

$$f(S) = \{ t | \exists s \in S (t=f(s))\}$$
