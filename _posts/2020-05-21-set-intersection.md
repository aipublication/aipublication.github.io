Intersection of two [Sets]({% post_url 2020-07-03-sets %}) $A$ and $B$ is the set that contains elements common in both sets.

































































































































































































































































































































































It is denoted as $A \cap B$.


































































































































































































































































































































































$$A \cap B = \{ x | (x \in A) \wedge (x \in B)\}$$
